# Assignment

A responsive web application built with Vite.

🔗 **Live Demo:** [https://css-assignment-tan.vercel.app/](https://css-assignment-tan.vercel.app/)

---



### Installation & Development

Clone the repository and start the development server:

```bash
git clone https://github.com/def-life/css-assignment.git
cd css-assignment
npm run app:dev
```

The app will be available at `http://localhost:5173` by default.
